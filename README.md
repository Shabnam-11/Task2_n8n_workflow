I use the coinGecko API because it provides free & real time cryptocurrency market data, including prices, market cap, and daily changes. I chose it because it is reliable and doesn't need API keys. After fetching the top 10 coins, we transformed the data using a code node to keep only the top 5 results, ensuring we focus on the most relevant coins. Next, we enriched the top coin (Bitcoin) with a second API call to retrieve detailed statistics, such as its full market history. If any API call fails, we use an error trigger node: instead of the workflow silently failing, it logs the error and sends a fallback message. This ensures that every failure is caught and addressed, maintaining a robust and reliable alert system.
